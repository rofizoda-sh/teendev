CSS Profile

College Scholarship Service Profile - это онлайн-заявка на получение финансовой помощи, которая используется более чем 400 колледжами и университетами США. CSS Profile собирает подробную финансовую информацию о вас и вашей семье, чтобы определить вашу потребность в финансовой помощи.


Цель CSS Profile:

  • Определить вашу потребность в финансовой помощи на основе вашей финансовой ситуации
  • Рассмотреть вас для получения институциональных стипендий, грантов и другой помощи, помимо федеральной помощи, на которую вы можете иметь право

Кто должен подавать CSS Profile:

  • Студенты, подающие заявки на получение финансовой помощи в колледжи и университеты, которые требуют CSS Profile.
  • Международные студенты, подающие заявки на получение финансовой помощи в США.

Информация, собираемая в CSS Profile:

  • Доход и активы вашей семьи
  • Демографическая информация
  • Информация о домохозяйстве и расходах
  • Информация об образовании и занятости родителей/опекунов

Сроки подачи:

Сроки подачи CSS Profile варьируются в зависимости от университета. Обычно рекомендуется подавать заявку как можно раньше после 1 октября года, предшествующего поступлению в колледж


Плата за подачу заявки:

За подачу заявки на CSS Profile взимается плата в размере 25 долларов США. Для семей с низким доходом доступны варианты освобождения от платы.


Советы по заполнению CSS Profile:

  • Начните заполнять заявку заранее, чтобы у вас было достаточно времени для сбора необходимой информации.
  • Тщательно следуйте инструкциям и предоставляйте точную и полную информацию.
  • Обратитесь за помощью к финансовому консультанту или консультанту по поступлению в колледж, если вам нужна помощь в заполнении заявки.
  • Подайте заявку как можно раньше, чтобы максимально увеличить свои шансы на получение финансовой помощи.

Преимущества подачи CSS Profile:

  • Возможность получить финансовую помощь помимо федеральной помощи.
  • Рассмотрение для получения институциональных стипендий и грантов.
  • Более комплексная оценка вашей финансовой ситуации.
